# Social Profile Page

## Description

A frontend project that recreates a modern social networking profile page using **HTML5** and **CSS3**. The interface includes a profile card, education section, connection requests, connections list, and a responsive dashboard layout.

This project was developed to practice page layout, CSS positioning, Flexbox, card-based interfaces, and user interface design.

---

## Technologies Used

- HTML5
- CSS3

---

## Features

- Modern profile page layout
- Navigation bar
- User profile card
- Education section
- Connection requests panel
- Connections list
- Card-based UI design
- Responsive page structure
- Clean and organized styling
